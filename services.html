Kunalo Funnel

The Kunalo Funnel for software adoption is based on a modification of a marketing funnel. This type of funnel is often used in a marketing automation systems. Although the layers look similar. There are significant differences in how the software adoption funnel operates.

In a typical marketing campaign, the engagement is usually some form of online engagement such as registration to download a whitepaper. In a software adoption funnel, the engagement could be at a hackathon, on the comment section of a code distribution web page, or in a forum. Most significantly, the content used to trigger the download is usually a document that contains code examples.

The ability to provide great code examples on how to use the software is common problem that marketers have. Developers running software adoption campaigns usually run into problems with generating awareness.



Process
The Kunalo process is used to identify bottlenecks in software adoption. Kunalo will then develop specific campaigns to increase the use of the software development kit, API, or platform.
