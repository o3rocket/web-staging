--- layout: post title: Microsoft Windows Continues To Dominate Developer Usage date: 2013-12-10 author: Craig Oda comments: false --- Windows continues to be the main platform for software development in many industries and for many languages. {{excerpt_separator}}
In order to get a better idea of what platforms developers were using to create great software, I looked at different sources of data.

My first source was the Net Market Share desktop OS market share data from November, 2013.

This data indicates that Windows accounts for 90.73% of users.  Mac OS X accounts for 4.27% of users.  The Linux data is available on the Net Market Share site.  Linux accounts for 1.56% of users.

To get a better idea of developers, I looked at the Eclipse Community Survey Report from June 2013.


This indicates that Windows is used by 55% of software developers as their primary platform for software development.  Linux is a strong second with 35%.  Mac OS X trails significantly behind with 8.7% of developers in the 2013 survey.    Note that in the most recent survey, Linux gained at the expense of Mac OS X.

The reasons for the lack of popularity of Mac OS X for software developers using Eclipse became apparent when I looked at the target platform for deployment.


Windows is still used as the primary operating system for deployment.  

The software developers were primarily developing software and hardware high-tech products.

The results are more skewed toward Java developers.  Of the respondents, 67% were developing on Java and 11% were developing on C/C++.

The TIOBE Programming Community Index for December 2013 shows that C is used by 17.9% of developers and Java is used by 17.3%.  Adding up the developers of C, Objective-C, and C++, the total "C family" developers is over 37%.

The Eclipse Foundation report doesn't give a full view of all developers.  It focuses primarily on professional Java developers building products for software and hardware vendors.

To get another view of Python programmers, I looked at the traffic to my blog on teaching Python programming.  The blog is focused heavily on basic coding techniques.  The top traffic referral is Stackoverflow. 



Among people learning or teaching Python programming, 59% use Windows.   Although the blog doesn't cover Java development, the results are skewed toward people with Android phones since many of the lessons cover using Python with Pygame, and Pygame Subset for Android to get Python applications running on Android.

I have been trying to grab OS usage from either GitHub or StackOverflow directly.  If anyone knows how to do this, please leave a comment on this blog.

Other sources of data.
  • Steam Platform Survey.  This is obviously skewed heavily toward Windows since almost all games run on Windows and only a small percentage run on Linux and Mac OS X.  Even with this limitation, the Steam survey is useful in showing that if a person is trying to reach a desktop computer games developer, you need to target Windows developers.  If you're developing for Microsoft XBox, you're probably using Microsoft Windows.
  • Android versus iOS market share.   The iPhone has a huge impact on developer platforms.  Although Android is the dominant mobile platform, iOS has more gamers using it.  If you're developing a game for the iPhone, the developer is probably using Mac OS X.
